식별자 VS 키워드

키워드

자바스크립트가 처음 만들어질 때 정해진 특정한 기능들이 정의 되어있는 단어이다.


자바스크립트의 대표적인 키워드

var 변수를 선언한다.
function 함수를 선언한다.
return 함수의 결과값을 반환한다.
if ... else if문(조건문)을 실행한다.
do ... while do while반복문을 실행한다.
for for반복문 실행한다.
continue loop문을 빠져나가 가장상단으로 간다.
switch switch문을 실행한다.
break loop문이나 switch문을 종료하기 위해 사용한다.
주의 : 자바스크립트 키워드는 예약어로서, 변수명으로 사용 할 수 없다.

식별자

식별자는 자바스크립트에서 이름을 붙일 때 사용하는 단어이다. 식별자의 종류는 변수명, 함수명, 속성명, 메소드명이 있다.


식별자 표기법

1. 카멜 표기법(camelCase)
첫번째 단어는 소문자로 시작하고 두번째 단어는 대문자로 시작한다. 모양이 낙타를 닮아 카멜 표기법 이라고 부른다. 변수, 객체, 함수에 사용된다.

2. 파스칼 표기법(PascalCase)
첫번째 단어와 두번째 모두 대문자로 시작한다. 함수명, 클래스명, 생성자에 사용된다.

3. 스네이크 표기법(snake_case)
단어를 _(언더바)로 구분한다. 변수명, 함수명, 데이터 타입, 네임스페이스명 등에 사용된다.


식별자 선언 시 주의 사항

1. 변수명 첫 글자로는 $, _(언더바), 영문자만 올 수 있다.

var 1num=10; (X)
var $num=10; (O)

2. 변수명 첫 글자 다음은 영문자, 숫자, $, _(언더바)만 포함할 수 있다.

var 100num=10; (X)
var num100=10; (O)

3. 변수명으로는 예약어(document, location, window 등)를 사용할 수 없다. 예약어란 이미 자바스크립트에서 사용중인 단어를 말한다.

var document=10; (X)
var num=10; (O)

4.변수명을 지을 때는 되도록 의미를 부여해 작성하는 것이 좋다.

var num="hello"; (X)
var num=10; (O)

5. 변수명을 사용할 때는 대소문자를 구분해야 한다.

var num=10; document.write(Num) (X)
var num=10; document.write(num) (O)
